# 01. Философия: почему Spec-First

Backlinks: [[README]]

## Главная идея
Не писать код, пока проект не описан спецификацией. Для AI это критично: без контекста агент генерирует фрагменты, а не систему.

## Что дает подход
- Больше автономности AI-агента.
- Меньше конфликтов между модулями.
- Меньше переделок и ручного микроконтроля.

## Ключевой принцип
Спецификация - это ТЗ для AI-агентов. Чем конкретнее таблицы, эндпоинты, компоненты и edge cases, тем меньше уточнений и выше качество результата.
