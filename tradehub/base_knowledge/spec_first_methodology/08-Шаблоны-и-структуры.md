# 08. Шаблоны и структуры

Backlinks: [[README]] [[10-Чеклист-готовности]]

## Принцип
Выходные артефакты должны быть copy-paste ready, без заглушек и недоописанных блоков.

## Шаблон спецификации фичи
- Описание.
- User Stories.
- Модель данных.
- API.
- Экраны.
- Бизнес-логика.
- Edge cases.
- Приоритет и зависимости.

## Рекомендуемая структура репозитория
- `CLAUDE.md`
- `SPEC_TEMPLATE.md`
- `.claude/agents/`
- `.claude/rules/`
- `.claude/skills/`
- `src/`
- `supabase/migrations/`
- `supabase/functions/`
