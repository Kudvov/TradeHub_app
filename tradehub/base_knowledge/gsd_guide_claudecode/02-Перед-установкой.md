# 02. Что нужно перед установкой

Backlinks: [[README]] [[03-Установка-GSD]]

## Обязательно
- VS Code или Cursor (актуальная версия).
- Расширение Claude Code.
- Подписка Claude (Pro/Max/Team/Enterprise).
- Node.js 18+.

## Проверка Node
```bash
node --version
```

## Важно
Команды установки `npx ...` вводятся в терминал, а не в чат Claude Code.
