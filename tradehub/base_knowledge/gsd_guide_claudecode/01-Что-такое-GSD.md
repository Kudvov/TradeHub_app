# 01. Что такое GSD

Backlinks: [[README]]

## Определение
Get Shit Done (GSD) — методика контекст-инжиниринга и spec-driven разработки для Claude Code.

## Проблема, которую решает
`context rot`: по мере роста диалога AI теряет системный контекст и начинает генерировать несогласованный код.

## Что дает GSD
- структурированную спецификацию;
- планирование по фазам;
- автономное исполнение через субагентов;
- верификацию результатов;
- сохранение контекста между сессиями.

## Что устанавливается
Набор markdown-конфигов в `.claude/` (команды, агенты, ядро, hooks), без тяжелых бинарников.
