# 03. Установка GSD

Backlinks: [[README]] [[02-Перед-установкой]]

## Варианты
### Глобально (рекомендуется)
```bash
npx get-shit-done-cc --claude --global
```

### Локально (на один проект)
```bash
npx get-shit-done-cc --claude --local
```

## Проверка после установки
В чате Claude Code:
```text
/gsd:help
```
Если список команд появился — установка успешна.
