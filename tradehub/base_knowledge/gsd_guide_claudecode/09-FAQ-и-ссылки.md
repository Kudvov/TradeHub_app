## FAQ
- GSD работает через VS Code extension Claude Code.
- Команды установки (`npx ...`) всегда в терминале.
- Свои субагенты в `.claude/agents/` не должны перезаписываться GSD.
- Для обновления обычно повторяют команду установки.
