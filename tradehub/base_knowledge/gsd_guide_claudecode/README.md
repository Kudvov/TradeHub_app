## Кратко
GSD — система контекст-инжиниринга/spec-driven разработки для Claude Code, которая борется с context rot через структурирование спецификаций, фаз и субагентов.

## Как использовать в Teleposter
- Для больших изменений: `new-milestone -> plan-phase -> execute-plan`.
- Для быстрых правок: `quick`.
- Для поддержки качества: регулярный `health` и review после фазы.
