# 08. Автоматизация и удаление GSD

Backlinks: [[README]]

## Автоматизация
- Настроить разрешения в `.claude/settings.json`.
- Выбрать режим: `Ask before edits` или `Auto-accept edits`.

## Удаление
### Удалить глобальную установку
```bash
npx get-shit-done-cc --claude --global --uninstall
```

### Удалить локальную установку
```bash
npx get-shit-done-cc --claude --local --uninstall
```
