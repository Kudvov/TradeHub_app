## Шаги
1. Создать vault в Obsidian.
2. Попросить Claude Code проанализировать кодовую базу и создать структуру заметок.
3. Добавить бизнес-контекст (опционально).
4. Добавить инструкции работы с vault в `CLAUDE.md`.
5. В финале сессии запускать ритуал: "сохрани сессию".

## Рекомендуемая структура
- `00-home/`
- `atlas/`
- `knowledge/` (integrations, decisions, debugging, patterns, business)
