# 05. Правило именования заметок

Backlinks: [[README]] [[06-Реальные-юзкейсы]]

## Принцип
Название заметки = утверждение, а не категория.

## Примеры
- Плохо: `auth.md`
- Хорошо: `авторизация через Supabase Auth с RLS на каждой таблице.md`

## Почему это важно
Claude сначала видит названия файлов. Утверждения в названиях помогают быстрее находить релевантную заметку.

## Связность
Использовать wiki-ссылки `[[...]]` внутри текста для построения графа знаний.
