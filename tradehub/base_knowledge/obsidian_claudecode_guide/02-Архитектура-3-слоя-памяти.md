# 02. Архитектура: 3 слоя памяти

Backlinks: [[README]] [[07-Структура-vault]]

## Слой 1: CLAUDE.md
Короткие правила, стек, архитектурные рамки. Читается автоматически при старте.

## Слой 2: Obsidian Vault
Долгосрочный граф знаний: решения, баги, бизнес, история.

## Слой 3: Memory Claude Code
Автозаметки в `~/.claude/projects/.../memory/` с предпочтениями и корректировками поведения.

## Взаимодействие
- `CLAUDE.md` указывает, что и когда читать в vault.
- Vault дополняет текущую сессию контекстом.
- Memory хранит персональные операционные заметки.
