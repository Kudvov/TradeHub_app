# Обзор проекта

## Назначение

**Teleposter** — веб-приложение на **SvelteKit** с **PostgreSQL**: показывает объявления по городам, источник данных — **публичные Telegram-группы** (каналы). Контент подтягивается скриптом синхронизации, который парсит HTML embed-страницы сообщений (`t.me/username/message_id`), без обязательного Telegram Bot API для основного потока.

## Пользовательские сценарии

- Главная и выбор города.
- Лента города: фильтры по категории, периоду (сегодня / вчера / …), поиск.
- Карточка объявления: фото, контакт, жалоба.
- Админка (`/admin/groups`): группы Telegram, парсер, логи `journalctl`, аналитика, жалобы (при включённой авторизации по cookie).

## Имя и пути

- В `package.json` имя пакета: **teleposter**.
- Репозиторий на диске может называться `TradeHub_app`; каталог приложения — **`tradehub/`**.
- На сервере после деплоя код лежит в **`/opt/teleposter`** (архив собирается из папки `tradehub`).
