# Обзор проекта

## Назначение

**Teleposter** — веб-приложение на **SvelteKit** с **PostgreSQL**: показывает объявления по городам, источник данных — **публичные Telegram-группы**. Контент подтягивается скриптом синхронизации, который парсит HTML embed-страницы сообщений (`t.me/username/message_id`), без обязательного Telegram Bot API.

## Пользовательские сценарии

- Главная и выбор города.
- Лента города: sidebar с фильтрами (категория, цена от/до, период), поиск по тексту (full-text PostgreSQL).
- Карточка объявления: фото, контакт, жалоба, Open Graph / JSON-LD.
- Админка (`/admin/groups`): группы Telegram, парсер, логи journalctl, аналитика, жалобы.
