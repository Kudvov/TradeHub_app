# База данных

## ORM

**Drizzle** + `drizzle-kit push` для продакшена (без обязательных файлов миграций в репозитории для каждого шага — по ситуации).

## Основные таблицы (см. `lib/server/db/schema.ts`)

| Таблица | Назначение |
|---------|------------|
| `cities` | Города, slug, `listings_count` |
| `categories` | Категории объявлений |
| `telegram_groups` | Группы TG: `username`, `city_id`, курсоры `last_message_id`, `start_message_id`, `is_active` |
| `listings` | Объявления: текст, цена, фото (json), связь с группой и городом |
| `listing_reports` | Жалобы пользователей |
| `banned_authors` | Забаненные контакты (проверка при синке/рефреше) |

## Seed

```bash
npm run db:seed
```

Заполняет города, категории, при необходимости стартовый набор групп — см. актуальный `seed.ts`.

## SQL вручную

В `tradehub/scripts/migrations/` лежат одноразовые SQL (баны, очистка данных и т.д.) — выполнять через `psql` на сервере при необходимости.
