# Стек и структура

## Стек

| Слой | Технология |
|------|------------|
| Фреймворк | SvelteKit 2, Vite |
| SSR / API | SvelteKit server routes, `adapter-node` для продакшена |
| БД | PostgreSQL, Drizzle ORM |
| Парсер | Node (`tsx`), HTTP к `t.me`, cheerio |
| Продакшен | systemd (`teleposter.service`, `teleposter-parser.timer`), nginx → порт 3000 |

## Каталог `tradehub/src`

| Путь | Назначение |
|------|------------|
| `routes/` | Страницы, `+page.server.ts`, API `routes/api/` |
| `lib/server/db/` | Схема Drizzle, seed |
| `lib/server/telegram/` | `sync.ts`, `scraper.ts`, `extractor`, рефреш фото, дедуп; вспомогательные скрипты `cleanup-duplicates.ts`, `refresh-images.ts` |
| `lib/components/` | UI-компоненты (часть стилей вынесена в `.css`) |
| `lib/server/parser-journal.ts` | Сборка текста лога для админки из `journalctl` |

## Статика

- `tradehub/static/` — favicon, логотип и т.д.

## Тесты

- `npm run test` — unit-тесты (extractor, дедуп, стоп-слова и др.)
