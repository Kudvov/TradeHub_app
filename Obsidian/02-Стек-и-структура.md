# Стек и структура

## Стек

| Слой | Технология |
|------|------------|
| Фреймворк | SvelteKit 2, Svelte 5 (runes), Vite |
| SSR / API | SvelteKit server routes, `adapter-node` |
| БД | PostgreSQL, Drizzle ORM |
| Парсер | Node (`tsx`), HTTP к `t.me`, cheerio |
| i18n | svelte-i18n (ru / en / ka) |
| Мониторинг | Sentry (`@sentry/sveltekit`) — server + client |
| CI/CD | GitHub Actions → SSH deploy |
| Продакшен | systemd, nginx → порт 3000 |

## Каталог `tradehub/src`

| Путь | Назначение |
|------|------------|
| `routes/` | Страницы и API |
| `routes/[city]/` | Лента города: sidebar layout, фильтры, пагинация |
| `routes/listing/[id]/` | Карточка объявления + OG/JSON-LD |
| `routes/sitemap.xml/` | Динамический sitemap |
| `routes/api/` | REST: listings, cities, categories (с rate limiting) |
| `lib/server/db/` | Схема Drizzle, seed, mock-data |
| `lib/server/telegram/` | `sync.ts`, `scraper.ts`, extractor, рефреш фото, дедуп |
| `lib/server/rate-limit.ts` | In-memory sliding window rate limiter |
| `lib/components/` | CategoryFilter, PeriodSelect, ListingCard, … |
| `hooks.server.ts` | Sentry `handleError` |
| `hooks.client.ts` | Sentry client init |

## Ключевые файлы фильтров (лента города)

- `routes/[city]/+page.svelte` — разметка: sidebar (категории вертикально, форма цены, период) + grid объявлений
- `routes/[city]/+page.server.ts` — загрузка: фильтры по категории, цене (`::numeric` cast), периоду, full-text search
- `routes/[city]/city-page.css` — CSS grid (`220px 1fr`), mobile-first
