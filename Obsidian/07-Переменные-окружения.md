# Переменные окружения

Полный комментированный список — **`tradehub/.env.example`**. Ниже сжато.

## Обязательные на проде

| Переменная | Назначение |
|------------|------------|
| `DATABASE_URL` | PostgreSQL connection string |
| `ADMIN_KEY` | Пароль для входа в админку (`/admin/groups`) |
| `ADMIN_LOGIN` | Логин для админки (по умолчанию `admin`) |

## Мониторинг

| Переменная | Назначение |
|------------|------------|
| `SENTRY_DSN` | DSN для Sentry (server-side) |
| `PUBLIC_SENTRY_DSN` | DSN для Sentry (client-side, публичная) |

## Парсер и сеть

| Переменная | Назначение |
|------------|------------|
| `PARSER_HTTPS_PROXY` / `HTTPS_PROXY` | Прокси для запросов к `t.me` |
| `SYNC_MESSAGE_GAP_MS` | Пауза между запросами при синке |
| `SYNC_PROBE_GAP_MS` | Пауза в поиске «верхнего» message id |
| `SYNC_CURSOR_FLUSH_EVERY` | Как часто писать курсор в БД |
| `REFRESH_IMAGES_AFTER_SYNC` | После синка вызывать рефреш картинок; `0` — отключить |

## Прочее

| Переменная | Назначение |
|------------|------------|
| `LISTINGS_PERIOD_TIMEZONE` | Зона для фильтров «сегодня/вчера» |
| `ORIGIN` | При проблемах с CSRF за reverse-proxy |
| `NODE_ENV` | `production` на проде (Sentry использует для окружения) |
